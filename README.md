# Jooeon Lee Blog

개인 개발 기록과 학습 내용을 정리하는 블로그입니다.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Contentlayer (MD/MDX)

## Local Development

```bash
yarn
yarn dev
```

Open http://localhost:3000

## Writing Posts

- Posts live under `data/blog`.
- Create `.md` or `.mdx` files with frontmatter.
- Author profile lives in `data/authors/default.md`.
